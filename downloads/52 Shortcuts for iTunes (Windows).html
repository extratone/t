52 Shortcuts for iTunes (Windows)

1. Playback shortcuts
Enter Play the currently selected song immediately
Ctrl+Alt+Arrow Right/Arrow Left Move forward or backward within a song
Shift+Arrow Right/Arrow Left Listen to the next or previous album in a list
Arrow Left/Arrow Right Go to the next / go to previous song in a list
2. Library and Playlist
Ctrl+click the checkbox next to a song Select or deselect all the songs in a list
Ctrl+Shift-N Create a new playlist from a selection of songs
Ctrl+Alt-N Create a new Smart Playlist
F5 Refresh the Radio list, when Radio is selected (or Genius Playlist if selected)
3. iTunes Store
Ctrl+]/[ Go to the next page / go to previous page in the iTunes Store
F5 Check for new podcast episodes (when Podcasts is selected, below Library)
Type in Search field and press Shift+Enter Initiate a search in the iTunes Store (from anywhere in iTunes)
Alt+s Open the Store menu
F5 Check for app updates (when Apps is selected, below Library)
4. File and Window
Ctrl+click a triangle Expand or collapse all the triangles in the Radio’s Stream list
Alt+N / Alt+P In the Get Info window, see the info for the next or previous song in the list
Ctrl+F Select the search field
Press ?, then key available See more options when a visual effect is showing
While the video is playing, Ctrl+click the movie, and choose Set Poster Frame Choose artwork for the selected video
Ctrl+U Stream audio file at a specific URL to iTunes
5. File Menu
Ctrl+N Create a new playlist
Ctrl+O Open / add a file to your music library
Ctrl+W Close the iTunes window
Ctrl+I Open the song or CD info window for the selected song or CD
Ctrl+R Show where a song file is located
Ctrl+L Show the currently playing song in the list
6. Edit Menu
Ctrl+, Open iTunes Preferences
Ctrl+Z Undo your last typing change while still in edit mode
Ctrl+X/Ctrl+C/Ctrl+V Cut the selected song’s information or artwork
Ctrl+A Deselect all the songs in the list
Ctrl+Shift+A Hide or show the Artist and Album columns
7. Control menu shortcuts
Space Start or stop playing the selected song
Arrow Right/Arrow Left Play the next song / play previous song in a list (When a song is playing)
Ctrl+Arrow Up/Arrow Down Increase the volume / decrease the volume
Ctrl+Alt+Arrow Down Mute the sound (song keeps playing)
Ctrl+E Eject a CD
8. View Menu
Ctrl+Shift+s Switch to the MiniPlayer
Ctrl+B Hide or show the Browser (Artist and Album columns)
Ctrl+T Turn the visualizer on or off
Ctrl+F Fullscreen visualizer
Ctrl+Shift+F See more options when a visual effect is showing
Ctrl+J Open the View Options window for the selected source
9. With full keyboard navigation enabled
Tab Select the next control
Enter Click the currently selected control
Arrow Left/ Arrow Right or +/- on the numeric keypad Open or close the selected item’s disclosure triangle
10. Other
Ctrl+? Open iTunes Help
Ctrl+Shift (while opening iTunes) Open iTunes in “safe mode” (without external plug-ins)
F2 Edit the name of the currently selected item
F5 Refresh source list
Ctrl+Break Interrupt the current process (burning a CD, syncing, converting a file, and so on)
Ctrl+Shift+Arrow Right/Arrow Left Go to the next or last chapter (during playing spoken words, if available)
Press and hold Control+Shift when connecting iPod/iPhone Prevent iPod / iPhone from automatically syncing when you connect it to your computer
Press ?, then key available See more options when a visual effect is showing
